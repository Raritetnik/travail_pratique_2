Commande Description Exemple
cd La commande cd vous permet de changer de répertoire (cd = change directory). Quand vous ouvrez un terminal en mode utilisateur vous êtes dans votre répertoire personnel (/home/utilisateur). cd /home/utilisateur/dossier ### vous déplacera à votre répertoire (/home/utilisateur/dossier)
ls La commande ls liste les fichiers et les sous-dossiers. (ls = list) ls -l ### ceci vous donnera la liste de vos fichiers et dossiers version longue.
ls La commande chmod permet de modifier les droits d'accès sur les fichiers ou répertoires. Seul le propriétaire des fichiers ou répertoires (ou root) peut faire cette manipulation. chmod ugoa|+/-/=|rwxugo fichier*
chown La commande chown permet de changer le propriétaire d'un fichier ou répertoire (chown = change owner). Seul le propriétaire du fichier (ou root) peut faire cette manipulation. chown utilisateur2 /home/utilisateur/doc.txt ### "utilisateur2" devient propriétaire du fichier doc.txt
